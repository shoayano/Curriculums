# カリキュラム管理アプリ

## 概要
学習塾において生徒ごとにカリキュラムを作成し、学習を管理するためのWebアプリケーションです。<br>
先生ユーザーは、塾で扱うテキストをベースに各生徒のカリキュラムの作成/編集、学習の管理を行うことができます。<br>
生徒ユーザーは、自身のカリキュラムを閲覧、学習の管理を行うことができます。<br>
<br>
<br>
