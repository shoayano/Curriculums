# カリキュラム管理アプリ

## 概要
学習塾において生徒ごとにカリキュラムを作成し、学習を管理するためのWebアプリケーションです。<br>
塾で扱うテキストをベースに、カリキュラムの作成/編集、学習の管理を行います。<br>
<br>
